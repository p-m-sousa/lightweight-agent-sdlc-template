# UI Tweaks

Use this lightweight list for small visual, layout, accessibility, and copy refinements noticed while using the product.

These are not product features. If a tweak grows into behavior, data, navigation, permissions, backup/restore, or larger workflow changes, move it into `BACKLOG.md` and the Kanban flow.

UI tweaks may be completed without the full feature workflow when they are clearly small, low-risk refinements that preserve the product vision.
